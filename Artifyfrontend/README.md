# Artify - Digital Art Marketplace

![Artify Hero Section](https://images.unsplash.com/photo-1547826039-bfc35e0f1ea8?auto=format&fit=crop&q=80&w=1200)

**Artify** is a premium digital marketplace connecting artists with art enthusiasts. It provides a seamless platform for discovering, buying, and selling original artworks, including paintings, calligraphy, sketches, and digital art. The platform is designed with a focus on aesthetics, user experience, and secure transactions.

---

##  fast Features

### Marketplace
- **Browse & Filter**: Explore thousands of artworks with filters for category, price, and ratings.
- **Search**: Real-time search functionality to find specific art or artists.
- **Artwork Details**: High-resolution view, detailed descriptions, and artist information.

###  User Roles & Dashboards
- **Artists**:
  - Upload and manage artwork portfolios.
  - View sales statistics and analytics.
  - customizable profile with bio and featured works.
- **Buyers**:
  - Secure shopping cart and checkout flow.
  - Order history tracking and status updates.
  - "Favorites" and wishlists (planned).
- **Admins**:
  - Platform overview and revenue tracking.
  - User management (Ban/Unban).
  - Content moderation (Reports).

###  Security & Tech
- **Authentication**: Secure Login/Register with role-based access control (RBAC).
- **Protected Routes**: Dashboard and Checkout areas secured for authenticated users.
- **Responsive Design**: Fully optimized for Desktop, Tablet, and Mobile.
- **SEO**: Dynamic meta tags and titles for better search engine visibility.

---

##  Technology Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS v3
- **Routing**: React Router v6
- **State Management**: React Context API
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Utilities**: clsx, tailwind-merge

##  Installation & Setup

Follow these steps to run the project locally.

### Prerequisites
- Node.js (v16.0.0 or higher)
- npm or yarn

### Steps

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/yourusername/artify-frontend.git
    cd artify-frontend
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    ```
    The app will run at `http://localhost:5173`.

4.  **Build for Production**
    ```bash
    npm run build
    npm run preview
    ```
